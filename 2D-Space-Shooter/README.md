# 2D Space Shooter

Przeglądarkowa gra 2D typu Space Shooter stworzona w Three.js.

## Uruchamianie

Otwórz `index.html` w przeglądarce lub uruchom lokalny serwer:

```bash
python3 -m http.server 8000
```

## Sterowanie

- WASD / Strzałki - ruch statku
- SPACJA - strzał
- ESC - pauza
